# 🎬 ТСН АвтоРеклама (YouTube Studio Ad Automator)

Професійне розширення для Google Chrome, створене для автоматизації розстановки рекламних пауз у YouTube Studio. Ідеальний інструмент для новинних каналів та контент-мейкерів, що дозволяє значно прискорити процес монетизації.

---

## 🌟 Ключові можливості

### 1. 🔍 Розумний аналіз пауз
Бот ("Валера") автоматично перехоплює аудіо-дані з YouTube Studio та аналізує вейвформу. Він знаходить реальні моменти тиші, щоб реклама не переривала ведучого на середині слова.

### 2. 📈 Стратегія "Focus Start" (Утримання)
Унікальна функція, що дозволяє масово скупчувати рекламу на початку відео. 
*   **Максимізація доходу:** Більше реклами там, де утримання аудиторії найвище.
*   **Жорсткий Gap:** Попри скупчення, бот суворо дотримується заданої відстані між рекламами (наприклад, 1.5 - 2 хв), щоб не перевтомити глядача.

### 3. 🎯 Точність до кадру (Frames)
Ми не використовуємо "мілісекунди", які YouTube Studio часто розуміє хибно. Бот вводить час у форматі `ММ:СС:КК` (кадри 0-24), що гарантує потрапляння мітки піксель-у-піксель за вашим планом.

### 4. 🧲 Магнітне додавання
Якщо ви вирішите додати мітку вручну (кліком по вейвформі):
*   Бот автоматично "примагнітить" ваш клік до найближчої реальної паузи (якщо вона поруч у межах 2.5с).
*   Це забезпечує ідеальну вставку навіть при ручній корекції.

### 5. 🛡️ Обхід "00:00:00" багу
Спеціально розроблений механізм імітації дій людини (Backspace + typing) обходить React-захист YouTube Studio, який часто скидає першу мітку на нуль.

---

## ⚙️ Технічні особливості

*   **Огинаюча (Waveform):** Використовує O(N) алгоритм згладжування для швидкої обробки довгих відео (понад 1 год).
*   **Кешування:** Аудіо-дані завантажуються лише один раз. Зміна налаштувань перераховує позиції миттєво без повторного завантаження.
*   **Cleanup System:** Розумне керування ресурсами — скрипт автоматично чистить пам'ять та відключає Observer-и, якщо ви переходите на іншу сторінку.

---

## 🚀 Як встановити

1.  Завантажте цей репозиторій як ZIP-архів та розпакуйте його.
2.  Перейдіть у Chrome: `chrome://extensions/`.
3.  Увімкніть **"Developer mode"** (Режим розробника) у правому верхньому куті.
4.  Натисніть **"Load unpacked"** (Завантажити розпаковане) та виберіть внутрішню папку проєкту.
5.  Відкрийте редактор рекламних пауз у YouTube Studio — панель з'явиться автоматично.

---

## 🛠️ Використання

1.  Натисніть червону кнопку **"Аналізувати паузи"**.
2.  Налаштуйте кількість міток (наприклад, 20) та бажаний Gap (відстань).
3.  Натисніть **"Вставити маркери"**.
4.  Для повного очищення або перезапуску використовуйте кнопку `↻` у шапці панелі.
